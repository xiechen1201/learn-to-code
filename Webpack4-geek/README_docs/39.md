# 冒烟测试介绍和实际应用

冒烟测试是指提交测试的软件在进行详细深入的测试之前而进行的预测试，这种预测试的主要目的是暴露导致软件需要重新发布的基本功能失效等严重问题。

需要关注：

1、构建是否成功

2、每次构建完成后目录是否含有内容输出

    是够含有 js、css 静态资源文件

    是够有 HTML 文件
